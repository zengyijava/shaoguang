
执行步骤：

1.连接数据库:
IP：192.169.7.232，用户：root  ,密码：123456：
mysql -h192.169.7.232 -uroot -p123456 -P3306

2/选择数据库：
use empf;

3..首先查询当前库的版本
  在库中执行   SELECT VERSION from LF_DB_SCRIPT ORDER BY ID DESC;
  查询结果第一条即当前版本号。
4.执行升级脚本
 通过1步骤得到当前版本号，找到对应版本升级到最新版本的脚本文件依次执行。

 从标准版V7.1开始脚本的命名按照VXX.XX的格式，左边代表标准版大版本，右边每次有脚本更新依次加一
 比如标准版V7.1的脚本是V71.00，标准版V7.2的脚本是V72.00


依次执行升级脚本，如：
source /usr/local/sql/7300/LINUX系统标准版EMP实时库MYSQL升级脚本V73.06-V73.07.sql
